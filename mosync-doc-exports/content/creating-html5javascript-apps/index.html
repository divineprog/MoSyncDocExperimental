The MoSync SDK and MoSync Reload make it easy to develop apps for iOS, Android and other mobile platforms using familiar technologies like HTML5 and JavaScript. Your apps can access device features like  cameras and sensors. You can even create completely native user interfaces using our Wormhole JavaScript Library!

MoSync Reload makes mobile development a breeze. Use your favorite text editor, edit your HTML/JavaScript code, press a button and instantly see your changes in action on as many devices as you like.

The tutorial Getting Started with HTML5 and JavaScript show how to use the MoSync SDK to get an app up and running on your device right away!

The MoSync SDK is a full cross-platform mobile development environment for creating everything from enterprise applications to advanced 3D games using OpenGL. You can use it for everything from just packaging your Reload applications to creating HTML5/C++ hybrid applications that use both C/C++ and JavaScript.

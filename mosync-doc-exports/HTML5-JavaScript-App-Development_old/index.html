What is Reload?

Reload is a whole new way to develop mobile web applications using HTML5 & JavaScript, it gives you unparalleled access to the native functionality of mobile devices and instant results that you'll see directly on your phone live, just by hit the reload button on your browser and see the results on your iPhone, Android or Windows Phone. Imagine accessing all of the phones functions directly from JavaScript. Now you can!

Reload is available for direct download for Windows, Mac OS X and Linux (or you can keep reading to find out more).

Reload is a free and open source product (with no catches) and has been created by the leading open source team who gave you the MoSync SDK. If you like this you can support us by sharing this page with your friends (using the share box on the top right of the screen).

Why should I use Reload?

Here are some of the benefits of Reload:

  • Free and Open Source
  • Easy to use, designed for HTML5/CSS and JavaScript developers.
  • Works with all mobile JS frameworks, JQMobile, Sencha, Dojo, Zepto, etc.
  • Tutorials and guides to help you get started.
  • Fast turnaround cycle (no lengthy compilation).
  • Unparalleled access to native functionality.
  • Works seamlessly on iPhone, Android and Windows Phone.
  • Use your favourite editor.
  • Easy to install, works in Mac, PC and Linux.

Reload gives you can access native features such as camera, SMS, file system and sensors and create user interfaces using real, native UI controls.

Our Wormhole Libraries provide communication between the HTML5/JavaScript and the C/C++ code layers of your app. On the HTML5 side there are many great functions in the Wormhole JavaScript Library that you can call from your JavaScript. Similarly, on the C/C++ side, there is a full Wormhole C++ Library. Both libraries give you access to the features and capabilities or the device your app is running on. Communication between the two libraries is handled by the Wormhole Bridge.

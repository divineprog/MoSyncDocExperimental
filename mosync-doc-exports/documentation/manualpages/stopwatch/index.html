This example app is a stopwatch/lap timer built entirely with MoSync Reload. It makes use of the NativeUI JavaScript API which is part of the MoSync Wormhole JavaScript Library. The source code is extremely well commented so that you can see what is happening at each step.

This application uses the MoSync NativeUI JavaScript API, and works on all platforms supported by this library (see Feature/Platform Support).

This example is included in the MoSync SDK installation in the /examples folder. For information on importing the examples into your workspace, see Importing the Examples.

Behaviour

Simple stopwatch application showcasing various features of the Mosync NativeUI, including labels, layouts and buttons. The application consists of a main screen, with a timer, a list of lap times, and four buttons:

  • [img_assist|nid=8943|title=|desc=|link=none|align=none|width=32|height=32] — starts the incrementing timer.
     
  • [img_assist|nid=8944|title=|desc=|link=none|align=none|width=32|height=32] — pauses the timer.
     
  • [img_assist|nid=8945|title=|desc=|link=none|align=none|width=32|height=32] — records the lap time. Successive lap times are listed in the middle of the screen.
     

  • [img_assist|nid=8946|title=|desc=|link=none|align=none|width=32|height=32] — zeros the clock. Timer continues if it was running.

In the code

The application is broken down into two major parts:

Javascript functions and the HTML structure. The Javascript is also divided into files, user interface and timer functions, for easier comprehension.

  • Index.html walks through the steps of setting up the NativeUI, from initialization to display and then finally to interaction. HTML is used to structure the applications various elements using standard <div> tags with custom attributes. While the javascript handles more complicated task like determining the correct font size.
     
  • Stopwatch.js contains the stopwatch specific functions like start, pause, round and reset, as well as time and string formatting.

These instructions describe how to build the MoSync runtimes for Moblin. The runtimes can then be included when you build the MoSync SDK for Windows or build the MoSync SDK for OS X.

To build the Moblin runtimes you will need a machine running Ubuntu (9.10).

1. First install the runtime dependencies in the Linux terminal:

{syntaxhighlighter brush: cpp;fontsize: 100; first-line: 1; }sudo apt-get install gcc g++ ruby rake subversion rpm sudo apt-get install libgtk2.0-dev libexpat1-dev sudo apt-get install libbluetooth3-dev libsdl1.2-dev sudo apt-get install libsdl-image1.2-dev libsdl-ttf2.0-dev sudo apt-get install libfreeimage-dev libssl-dev{/syntaxhighlighter}

Important! Do not install libsdl-sound -- Moblin does not support this library.

2. Download the source tarball (.tar.bz2) of our current feature release from our website or from our open-source project repository:

  • MoSync website: http://www.mosync.com/download
  • Git Project Repository: http://code.google.com/p/mosync/downloads/list

Alternatively, get the source tarball for our latest nightly build from our website:

  • MoSync website: http://www.mosync.com/nightly-builds

or through TortoiseGit:

{syntaxhighlighter brush: cpp;fontsize: 100; first-line: 1; }git clone https://github.com/MoSync/MoSync.git MoSync git clone https://github.com/MoSync/Eclipse.git Eclipse{/syntaxhighlighter}

Important! You must download the same version as you downloaded to build the MoSync SDK for Windows or OS X. If you do not, when you try to use the SDK you may get an “IDL Version Mismatch” error.

3. Run the Linux runtime builder script in the MoSync source directory::

{syntaxhighlighter brush: cpp;fontsize: 100; first-line: 1; }/tools/RuntimeBuilder/linux/build-linux-runtime-package.sh{/syntaxhighlighter}

4. You will now have two files in the current directory:

{syntaxhighlighter brush: cpp;fontsize: 100; first-line: 1; }runtime.ubuntu.r*.i386.dbg.tar.gz runtime.ubuntu.r*.i386.rel.tar.gz{/syntaxhighlighter}

Rename them to:

{syntaxhighlighter brush: cpp;fontsize: 100; first-line: 1; }runtime.dbg.tar.gz runtime.rel.tar.gz{/syntaxhighlighter}

5. If you are building the MoSync SDK for Windows, move the files to the following folder on the Windows build machine,

{syntaxhighlighter brush: cpp;fontsize: 100; first-line: 1; }c:\mb\MoSyncReleasePackage\profiles\runtimes\moblin\1\{/syntaxhighlighter}

You are then ready to run the SDK build script.

If you are building the MoSync SDK for OS X, copy the runtimes to the machine on which you have installed the MoSync SDK, in the directory:

{syntaxhighlighter brush: cpp;fontsize: 100; first-line: 1; }/Applications/MoSync/profiles/runtimes/moblin/1/{/syntaxhighlighter}

6. Done.
